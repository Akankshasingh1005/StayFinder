# StayFinder 🏠

StayFinder is a full-stack web application inspired by Airbnb. It allows users to create listings, browse places to stay, make bookings, and manage reservations — all in one place.

## 🔧 Tech Stack

- **Frontend**: React, Tailwind CSS  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB (via Mongoose)  
- **Authentication**: JWT with HTTP-only cookies  
- **Image Storage**: Cloudinary
  
## ✨ Key Features

- User Sign Up, Login, Logout (with token-based auth)
- Create and manage property listings with photos
- Book places with date range selection
- View your bookings and booking requests
- Search bar for filtering properties
- Responsive design and clean UI
